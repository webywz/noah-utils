# Noah Utils

Noah Utils 是一个用于处理树结构和货币格式化的实用工具库。它提供了一系列函数来简化树节点的添加、删除、更新、扁平化以及获取节点深度等操作。

## 功能

- **树操作**
  - `addTree`: 向树中添加新节点。
  - `delTreeNode`: 从树中删除指定节点。
  - `updateTreeNode`: 更新树中指定节点的信息。
  - `flattenTree`: 将嵌套的树结构扁平化。
  - `getNodeDepthById`: 获取树中指定节点的深度。

- **数字操作**
  - `formatCurrency`: 格式化数字为货币字符串。
